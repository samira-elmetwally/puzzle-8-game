# AI 8-Puzzle Game

## Project Description
This project implements the 8-puzzle game using Python.
It includes:
- A manual image-based puzzle
- An AI solver using BFS algorithm

## AI Technique
The AI part uses Breadth First Search (BFS) to find the shortest path to the solution.

## Technologies Used
- Python
- Tkinter (GUI)
- Pillow (Image Processing)

## How to Run
1. Install requirements:
pip install pillow

2. Run:
python main.py
